# SecondPractice

이 폴더는 Windows API를 이용한 기본적인 그림 그리기 연습 프로젝트입니다.

## 프로젝트 설명

이 예제 프로그램은 윈도우를 생성한 뒤, 마우스를 눌러 드래그하면 화면에 선을 그릴 수 있는 간단한 Win32 프로그램입니다. 마우스 입력 처리와 그리기 동작의 기본 원리를 익히는 데 초점을 두고 있습니다.

## 주요 기능

- 윈도우 생성 및 표시
- 마우스 왼쪽 버튼 누름 시 그리기 시작
- 마우스 이동 중 선 그리기
- 마우스 버튼을 떼면 그리기 종료
- 기본적인 Paint 메시지 처리

## 파일 구성

- draw.cpp: 윈도우 생성, 메시지 처리, 마우스 기반 선 그리기 로직이 포함된 메인 소스 파일입니다.
- SecondProject.sln: Visual Studio 솔루션 파일입니다.
- SecondProject.vcxproj: Visual Studio 프로젝트 파일입니다.

## 실행 방법

1. Visual Studio에서 SecondProject.sln 파일을 엽니다.
2. 빌드 메뉴에서 솔루션 빌드를 실행합니다.
3. 실행하면 윈도우가 열리고, 마우스를 드래그하여 선을 그릴 수 있습니다.

## 참고

이 프로젝트는 MFC가 아닌 순수 Windows API 기반 예제이며, 기본적인 윈도우 메시지 처리와 GDI 그리기 동작을 이해하는 데 유용합니다.
