# About this Repository

이 저장소는 MFC(Microsoft Foundation Classes)를 학습하면서 만든 여러 실습 프로젝트를 모아둔 개인 학습용 저장소입니다.

## 학습 목적
- MFC 기반 GUI 애플리케이션의 구조 이해
- 다이얼로그 기반/SDI/MDI 애플리케이션 작성 경험
- 소켓 통신, 이미지 처리, 컨트롤 사용 등 실무 기능 실습

## 주요 학습 내용
- **MFC 다이얼로그, 뷰(View), 프레임(Frame)** 간 관계
- **메시지 맵**을 통한 이벤트 처리
- **GDI/GDI+**를 이용한 그래픽/그리기
- **소켓 통신**(클라이언트/서버)
- **컨트롤**(리스트 박스, 버튼, 툴바 등)

## 사용 방법
1. `*.sln` 파일을 Visual Studio에서 엽니다.
2. 디버그 또는 릴리스 모드로 빌드합니다.
3. 실행하여 기능을 확인합니다.
