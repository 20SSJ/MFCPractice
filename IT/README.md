# IT

이 폴더는 MFC 기반의 문서/뷰 구조를 활용한 이미지 표시 예제 프로젝트입니다.

## 프로젝트 설명

이 프로젝트는 MFC 애플리케이션의 기본 구조를 이해하기 위해 만들어진 예제입니다. 문서, 뷰, 프레임 창이 연결된 MDI(다중 문서 인터페이스) 형태로 구성되어 있으며, 사용자가 이미지를 열거나 표시할 수 있는 기본 기능을 포함합니다.

## 주요 특징

- MFC 기반의 문서/뷰 아키텍처 사용
- MDI 프레임 구조 지원
- 이미지 파일 로딩 및 화면 표시
- 마우스 클릭 위치에 이미지 출력
- 기본적인 메뉴, 컨텍스트 메뉴, 인쇄 기능 포함

## 파일 구성

- IT.cpp: 애플리케이션 초기화 및 기본 메시지 처리
- MainFrm.cpp / MainFrm.h: 메인 프레임 창 구현
- ITDoc.cpp / ITDoc.h: 문서 클래스 구현
- ITView.cpp / ITView.h: 뷰 클래스 구현
- BmpShowView.cpp: 이미지 표시 관련 동작을 담당할 가능성이 있는 소스
- IppDib.cpp / IppDib.h: 이미지 로딩 및 화면 렌더링 처리
- IT.rc: 리소스 정의
- IT.reg: 레지스트리 등록 정보

## 실행 방법

1. Visual Studio에서 IT.sln 파일을 엽니다.
2. 솔루션을 빌드합니다.
3. 실행하면 MFC 기반의 메인 창이 열리고, 이미지 관련 동작을 확인할 수 있습니다.

## 참고

이 프로젝트는 MFC의 기본 구조와 이미지 표시 기능을 학습하는 데 적합한 예제입니다. 이미지 파일이 필요할 경우 프로젝트 폴더 또는 적절한 경로에 이미지 파일을 준비해야 합니다.
