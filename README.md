# MFCPractice

## 📂 프로젝트 구조
각 폴더는 하나의 MFC 프로젝트입니다.

- `ChatClient/` : MFC 기반 클라이언트 채팅 애플리케이션
- `ChatServer/` : MFC 기반 서버 채팅 애플리케이션
- `FirstPractice/` : MFC 기본 실습 프로젝트 (간단한 그리기 등)
- `IT/` : 이미지 처리 관련 MFC 프로젝트
- `ListBox/` : 리스트박스, 툴바, 메뉴 등의 UI 컨트롤 실습
- `MsgBox/` : 메시지 박스 및 다이얼로그 실습
- `Network/` : 네트워크 소켓/통신 관련 실습
- `SecondPractice/` : 추가 실습 프로젝트 (그리기 등)

## 🧰 개발 환경
이 프로젝트들은 **Visual Studio (Windows)**에서 빌드 및 실행하도록 설계되었습니다.

- MFC 라이브러리를 포함하는 Visual Studio 버전(예: Visual Studio 2022)
- 솔루션 파일(`*.sln`)을 열고 빌드하면 됩니다.

## 🚀 시작하기
1. Visual Studio를 열고, 원하는 폴더의 `.sln` 파일을 엽니다.
2. 솔루션 구성(Release/Debug)과 플랫폼(Win32/x64)을 설정합니다.
3. 실행(F5) 또는 빌드(Ctrl+Shift+B)합니다.
