# Network 프로젝트

## 개요
`Network` 프로젝트는 MFC 기반 네트워크 서버 소켓 실습 예제입니다. 이 프로젝트는 TCP 소켓을 이용한 클라이언트 연결 처리, 메시지 송수신, 파일 수신 기능을 포함합니다.

## 주요 기능
- 서버 소켓 생성 및 바인딩
- 클라이언트 연결 수락 및 연결 정보 표시
- 클라이언트로부터 닉네임 및 텍스트 메시지 수신
- 수신한 메시지 브로드캐스트
- 파일/이미지 전송 시 수신 파일 저장
- 서버 시작/중지 제어

## 주요 파일
- `Network.sln` / `Network.vcxproj` : 솔루션 및 프로젝트 파일
- `NetworkDlg.cpp` / `NetworkDlg.h` : 메인 대화상자 구현
- `Server.cpp` / `Server.h` : 서버 소켓, 수신 쓰레드, 클라이언트 관리 구현
- `Network.cpp` / `Network.h` : 애플리케이션 엔트리 및 초기화

## 실행 방법
1. Visual Studio에서 `Network/Network.sln`을 엽니다.
2. 빌드 구성(Debug/Release)과 플랫폼을 선택합니다.
3. 실행(F5)합니다.

## 사용 방법
1. 서버 IP와 포트를 입력합니다.
2. `Start` 버튼을 클릭하여 서버를 시작합니다.
3. 클라이언트가 접속하면 대화상자에 클라이언트 정보가 표시됩니다.
4. 클라이언트가 전송한 텍스트 메시지는 수신 목록에 표시됩니다.
5. 파일/이미지 전송 시 `ReceivedFiles` 폴더에 저장됩니다.

## 참고 사항
- 현재 서버는 IPv4 전용으로 구현되어 있습니다.
- `ReceivedFiles` 폴더가 존재하지 않으면 서버가 자동으로 생성합니다.
- 프로젝트는 Windows 환경의 Visual Studio와 MFC 라이브러리를 필요로 합니다.
