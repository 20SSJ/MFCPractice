# FirstPractice

이 폴더는 Windows API를 이용한 기본적인 윈도우 프로그래밍 연습 프로젝트입니다.

## 프로젝트 설명

이 예제 프로그램은 간단한 윈도우를 생성하고, 키보드와 마우스 입력에 따라 화면에 다른 메시지를 표시하는 방식으로 Win32 프로그래밍의 기본 동작을 익히기 위한 코드입니다.

## 주요 기능

- 윈도우 생성 및 화면 표시
- 키보드 입력 처리
  - 키 누름 시 메시지 표시
  - 키 떼기 시 다른 메시지 표시
- 마우스 입력 처리
  - 좌클릭 위치 표시
  - 마우스 이동 시 안내 문구 표시
  - 더블클릭 시 메시지 박스 출력
- 기본적인 Paint 처리

## 파일 구성

- Practice.cpp: 윈도우 생성, 메시지 처리, 화면 출력 로직이 포함된 메인 소스 파일입니다.
- Practice.sln: Visual Studio 솔루션 파일입니다.
- Paint.vcxproj: Visual Studio 프로젝트 파일입니다.

## 실행 방법

1. Visual Studio에서 Practice.sln 파일을 엽니다.
2. 빌드 메뉴에서 솔루션 빌드를 실행합니다.
3. 실행하면 간단한 윈도우가 나타나고, 키보드/마우스 입력에 따라 동작합니다.

## 참고

이 프로젝트는 MFC가 아닌 순수 Windows API 기반 예제이므로, 기본적인 Win32 프로그래밍 구조를 이해하는 데 초점을 둔 연습용 코드입니다.
